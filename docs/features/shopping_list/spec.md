# Shopping List Specification

## 1. Overview
A simple checklist for ingredients to buy.

## 2. Requirements

- **Route**: `/shopping-list` (or `/shopping`)
- **Data Model**: `ShoppingItem` (`id`, `name`, `amount`, `checked`).
- **Features**:
  - Add item (Name, Amount).
  - Check/Uncheck item.
  - Delete item.
  - (Planned) Auto-add from Recipe ingredients.
