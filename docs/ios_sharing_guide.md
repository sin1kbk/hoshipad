# iOSでHoshipadにレシピを保存する方法

iOSの「ショートカット」アプリを使って、InstagramやSafariなどからHoshipadへレシピを簡単に保存できます。


## 初回設定(1回のみ)


### 1. ショートカットアプリを開く

iPhoneに標準搭載されている「ショートカット」アプリを開きます。


### 2. 新規ショートカットを作成

1. 右上の「+」ボタンをタップ
2. 「アクションを追加」をタップ

### 3. アクションを設定

以下の手順でアクションを追加します:

#### ステップ1: 共有シートに表示

1. 検索バーで「共有シート」と入力
2. 「共有シートに表示」をONにする

#### ステップ2: 入力を受け取る

1. 「アクションを追加」をタップ
2. 検索バーで「入力を受け取る」と入力して選択
3. 「テキスト」をタップして「URL」に変更

#### ステップ3: URLを取得

1. 「アクションを追加」をタップ
2. 検索バーで「URLを取得」と入力して選択

#### ステップ4: URLを開く

1. 「アクションを追加」をタップ
2. 検索バーで「URLを開く」と入力して選択
3. 表示されたURLフィールドをタップ
4. 以下のURLを入力:

   ```text
   https://hoshipad.vercel.app/add?url=
   ```

5. URLの末尾にカーソルを合わせ、「ショートカット入力」を選択

### 4. ショートカットに名前をつける

1. 上部の「新規ショートカット」をタップ
2. 「Hoshipadに保存」などの分かりやすい名前を入力
3. 右上の「完了」をタップ

## 使い方

### Instagramから保存

1. Instagramで料理の投稿を開く
2. 右上の「...」(メニュー)をタップ
3. 「リンクをコピー」をタップ
4. Safariなどのブラウザを開く
5. アドレスバーにペーストして開く
6. 共有ボタン(□に↑)をタップ
7. 「Hoshipadに保存」を選択
8. Hoshipadが開き、レシピ追加画面が表示されます

### Safariから保存

1. Safariでレシピページを開く
2. 共有ボタン(□に↑)をタップ
3. 「Hoshipadに保存」を選択
4. Hoshipadが開き、レシピ追加画面が表示されます

## トラブルシューティング

### 「Hoshipadに保存」が表示されない

1. 共有シートを下にスクロール
2. 「ショートカット」セクションを探す
3. それでも見つからない場合は、ショートカットアプリで「共有シートに表示」がONになっているか確認

### URLが正しく渡されない

1. ショートカットアプリを開く
2. 作成したショートカットを編集
3. 「URLを開く」アクションのURLが正しいか確認:

   ```text
   https://hoshipad.vercel.app/add?url=[ショートカット入力]
   ```


### Hoshipadアプリが開かない

- Safariでページが開きます(これは正常な動作です)
- Flutterアプリを直接起動することはできませんが、同じ機能が使えます

## その他の方法

### ブックマークレット(Safari専用)

Safariを使用している場合は、既存のブックマークレット機能も利用できます。詳しくはアプリ内のヘルプをご覧ください。
