# Product Overview

**Hoshipad** is a personal recipe management web application inspired by Cookpad. It aggregates recipes from multiple sources (YouTube, Instagram, Twitter, Cookpad) into a unified interface for easy browsing, searching, and management.

## Core Features

- **Recipe Management**: Add, view, edit, and delete recipes with images, ingredients, and notes
- **Multi-Source Support**: Import recipes from YouTube, Instagram, Twitter, and Cookpad
- **Cookpad Scraping**: Automated recipe import from Cookpad URLs
- **Authentication**: User accounts with email/password via Supabase Auth
- **Social Features**: Like recipes, view history, maintain shopping lists
- **iOS Integration**: Share sheet support for easy mobile recipe import

## Design Philosophy

- Cookpad-inspired orange theme with clean, responsive design
- Mobile-first approach with grid layouts
- Simple, intuitive UX focused on recipe discovery and management
