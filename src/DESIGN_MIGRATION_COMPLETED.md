## 🎨 デザイン統一

### グラデーション
```css
bg-gradient-to-r from-primary to-orange-500  /* ログインボタン */
bg-gradient-to-br from-primary to-orange-500 /* FAB、アバター */
```

### シャドウ
```css
shadow-lg hover:shadow-xl    /* ログインボタン、アバター */
shadow-2xl hover:shadow-primary/50  /* FAB */
shadow-md  /* 選択中のフィルターボタン */
```

### 丸みのあるボタン
```css
rounded-full  /* ナビゲーション、FAB、ログインボタン */
rounded-xl    /* カード、ドロップダウン */
rounded-lg    /* その他のボタン */
```

### カラー
```css
Primary: #FF7400 (Cookpad Orange)
Gradient: #FF7400 → #FF9500
Background: White, Gray-50, Gray-100
Text: Gray-900, Gray-700, Gray-500
```

## 🔄 Flutter版との主な違い

### 採用した違い
1. **アイコンライブラリ**: Material Icons → Lucide Icons
   - より軽量で、React/Next.jsとの統合が良好

2. **フォント**: Noto Sans JP → Inter (Next.jsデフォルト)
   - 後でGoogle Fontsを追加可能

3. **ロゴ**: 画像 → 絵文字
   - 簡易実装、後で画像に変更可能

### 保持した共通点
1. カラースキーム（オレンジ #FF7400）
2. グラデーションとシャドウの多用
3. 丸いボタンデザイン
4. 3つのタブナビゲーション
5. FABの配置と機能
6. ユーザーメニューのUI
