# デザイン移行 TODO リスト

## 🚨 重大な移行漏れ

### 1. ボトムナビゲーションバー ❌
**Flutter版**: 3つのタブ（ホーム、買い物、履歴）
**Next.js版**: なし

### 2. AppBarの機能 ❌
**Flutter版**:
- hoshipadロゴ + テキスト
- 共有ヘルプボタン (iOS共有)
- ブックマークレットボタン (📌)
- 通知ボタン

**Next.js版**:
- シンプルなヘッダーのみ
- ナビゲーションリンクが横並び
- 検索バーが別の場所

### 3. カテゴリー選択UI ❌
**Flutter版**: 丸いアイコンボタンの横スクロールリスト
**Next.js版**: テキストボタンのみ

### 4. FloatingActionButton (FAB) ❌
**Flutter版**: 右下の丸いボタン
**Next.js版**: ヘッダーのボタンのみ

### 5. ユーザープロフィールメニュー ❌
**Flutter版**:
- アバター表示
- ポップアップメニュー（ユーザー情報ヘッダー付き）
- プロフィール編集
- サインアウト

**Next.js版**: 簡易的なログアウトボタンのみ

### 6. グラデーションボタン ❌
**Flutter版**: オレンジのグラデーション + シャドウ
**Next.js版**: フラットなボタン

## 📋 詳細比較

### ヘッダー/AppBar

#### Flutter版の要素:
```
[ロゴ] hoshipad [Spacer] [共有] [📌] [🔔] [ユーザーメニュー/ログイン]
              [検索バー]
```

#### Next.js版の要素:
```
hoshipad [ホーム] [買い物リスト] [履歴] [ログイン] [レシピを追加]
```

### ナビゲーション

#### Flutter版:
- **ボトムナビゲーション**: ホーム、買い物、履歴
- **FAB**: レシピ追加（中央下）

#### Next.js版:
- **ヘッダーナビゲーション**: ホーム、買い物リスト、履歴
- **ヘッダーボタン**: レシピを追加（右上）

### カテゴリーUI

#### Flutter版:
```
[🥩] [🐟] [🥗] [🍚] [🍜] ...
肉料理 魚介  サラダ ご飯  麺類
```
丸いアイコンボタン、横スクロール

#### Next.js版:
```
[すべて] [YouTube] [Instagram] [Twitter] [クックパッド]
```
ソースフィルターのみ、カテゴリーなし

## 🎨 デザイン要素の違い

### 色とスタイル

#### Flutter版:
- グラデーションボタン (オレンジ #FF7400 → #FF9500)
- ボックスシャドウ多用
- Material Design 3
- 丸いボタン (border-radius: 20-24px)

#### Next.js版:

- シンプルなシャドウ

- 角が丸い (border-radius: 0.5rem)

#### Flutter版:
- Google Fonts: Poppins (ロゴ)
- Noto Sans JP (本文)

#### Next.js版:
- Inter (Next.jsデフォルト)
- システムフォント
